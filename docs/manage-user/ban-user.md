            <h2>Ban User</h2>
            <p>The "Ban User" feature allows admin to ban a certain user from using the platform due to violations.
           </p>
            <h2>Use Case Scenario</h2>
            <table border="1">
                <tr>
                    <td colspan="2" align="left">
                     Admin Bans User
                    </td>
                </tr>
                <tr>
                    <th>Actor(s)</th>
                    <td>User, Admin</td>
                </tr>
              <tr>
                <th>Goal</th>
                <td>To ban a user from using the platform.
              </td>
              </tr>  
                <tr>
                    <th>Precondtions</th>
                    <td>
                          1. The admin must be on the ban/restrict page.<br>
                    </td>
                </tr>
                <tr>
                    <th>Main Scenario</th>
                    <td>
                        1. The admin searches for the name of the user.
                        <br>
                        2. The admin can have the choice to either ban or restrict the user.
                        <br>
                        3. The admin clicks the ban button.
                        <br>
                    </td>
                </tr>
                <tr>
                    <th>Outcome: </th>
                    <td>The user can no longer access the platform.
                </td>
                </tr>
